# Restaurant Management System (WPF, MVVM)

Desktop application for managing a restaurant network, built with **WPF**, **MVVM**, and **Entity Framework Core**.

The project demonstrates clean architecture, separation of concerns, and practical usage of modern .NET technologies in a real-world scenario.

---

## Features

- Restaurant selection
- Employee management (add / remove / list)
- Menu management (add / remove / list)
- Reservation management
- SQL Server database with Entity Framework Core
- MVVM pattern with clean separation between View, ViewModel, and Data layers
